# 0x03. Shell, variables expansions

## Description
This project covers the concepts of Shell variables, expansions, and aliases. It explores how to create, manipulate, and use environment and local variables, as well as how to customize the shell environment.

## Tasks

| File | Description |
| --- | --- |
| 0-alias | Script that creates an alias `ls` with the value `rm *`. |

## Requirements
* All scripts are tested on Ubuntu 20.04 LTS.
* All scripts must be executable.
* The first line of every script must be `#!/bin/bash`.
